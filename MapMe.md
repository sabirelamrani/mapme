# Introduction #

MapMe basically displays 2D maps on Google Android but also adds additional features.


# Details #

These extra features are:
  * Interaction with Android GPS device (track GPS position, saves routes, etc)
  * Storage of location bookmarks/navpoints (including search capabilities) via db4o (http://developer.db4o.com)
  * Location sharing via Twitter, e-mail, SMS, etc

and more!
